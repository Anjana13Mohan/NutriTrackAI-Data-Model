# NutriTrackAI-Data-Model

Project Name: NutriTrackAI (MIBE Framework)

Goal: A relational data model designed to bridge clinical nutrition and diabetes risk assessment. 

Technical Stack: Google Sheets (Backend), Glide (No-code Logic Layer), GitHub (Data Versioning). 

Key Features:
•	Relational schema linking User Metadata with Time-Series Health Metrics.
•	Automated Clinical Logic for BMI, Blood Pressure risk stratification, and Glucose monitoring.
•	Normalized architecture to ensure data integrity for future ML model training.
